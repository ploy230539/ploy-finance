# 💰 Ploy Finance

แอปจัดการรายรับ-รายจ่ายส่วนตัว/ครอบครัว — ภาษาไทย, mobile-first, ติดตั้งเป็นแอปได้ (PWA)

## ฟีเจอร์

- 📝 บันทึก **รายรับ/รายจ่าย** + แก้ไขรายการ
- 🏷️ หมวดหมู่สำเร็จรูป + **สร้างหมวดเอง** (เลือก emoji + สี)
- 💳 **บัตรเครดิต/หนี้สิน** 5 ประเภท (บัตรเครดิต, สินเชื่อส่วนบุคคล, ผ่อนรถ, ผ่อนบ้าน, อื่นๆ) + ดอกเบี้ย + ติดตามงวด
- 👥 **หารบิล** ตามจำนวนคน + ติ๊กรับเงินคืน → บันทึกเป็นรายรับอัตโนมัติ
- 🧾 แนบ **สลิป/ใบเสร็จ** จากรูปถ่าย (บีบอัดอัตโนมัติ)
- 📅 ดู **รายเดือน** + ตั้ง **งบประมาณ** ต่อหมวด พร้อมเตือนเมื่อเกินงบ
- 🔁 **รายการประจำ** อัตโนมัติ (เงินเดือน/ค่าเช่า/ค่าสมาชิก)
- 💾 **สำรอง/กู้คืน** ข้อมูล (JSON) + ส่งออก Excel/CSV

## เทคโนโลยี

React 19 · Vite · Tailwind CSS v4 · React Router · PWA (Service Worker)

## เริ่มใช้งาน

```bash
npm install
npm run dev      # โหมดพัฒนา (localhost:5173)
npm run build    # build production
npm run preview  # ทดสอบ production build
```

ข้อมูลเก็บใน localStorage ของเบราว์เซอร์ (ออฟไลน์ได้)
